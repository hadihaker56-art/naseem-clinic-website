# طريقة رفع المشروع على GitHub من الجوال

هناك عدة طرق لرفع المشروع من الجوال:

## الطريقة 1: استخدام تطبيق GitHub الرسمي (الأسهل)

### على iPhone:
1. حمّل تطبيق **GitHub** من App Store
2. سجّل الدخول بـ:
   - البريد: `hadihaker56@gmail.com`
   - كلمة المرور: `hadi12345`
3. اضغط على **"+"** (إضافة)
4. اختر **"Create repository"**
5. ملء البيانات:
   - **Name**: `naseem-clinic-website`
   - **Description**: `عيادات النسيم السنية التجميلية`
   - **Visibility**: Public
6. اضغط **"Create"**
7. بعدها يمكنك إضافة الملفات من الجوال

### على Android:
1. حمّل تطبيق **GitHub** من Google Play
2. نفس الخطوات أعلاه

---

## الطريقة 2: استخدام موقع GitHub مباشرة من المتصفح

### الخطوات:
1. **افتح متصفح على جهازك الجوال**
2. **اذهب إلى**: `https://github.com`
3. **سجّل الدخول**:
   - البريد: `hadihaker56@gmail.com`
   - كلمة المرور: `hadi12345`
4. **اضغط على "+"** في الزاوية العلوية
5. **اختر "New repository"**
6. **ملء البيانات**:
   - Repository name: `naseem-clinic-website`
   - Description: `عيادات النسيم السنية التجميلية`
   - Visibility: Public
7. **اضغط "Create repository"**
8. **بعد الإنشاء**:
   - اضغط على **"Add file"** → **"Create new file"**
   - أو **"Upload files"** لرفع الملفات مباشرة

---

## الطريقة 3: استخدام تطبيق Git على الجوال

### تطبيقات موصى بها:

#### 1. **Termux** (Android فقط - متقدم):
- حمّل Termux من Google Play
- ثبّت Git: `apt install git`
- استخدم نفس أوامر الكمبيوتر

#### 2. **Working Copy** (iOS و Android):
- تطبيق متخصص لـ Git على الجوال
- يتيح لك إدارة المستودعات مباشرة
- سهل الاستخدام

#### 3. **GitJourney** (Android):
- تطبيق بسيط لإدارة Git

---

## الطريقة 4: الطريقة الأسهل - رفع الملفات مباشرة من المتصفح

### خطوات بسيطة جداً:

1. **افتح GitHub من متصفح الجوال**
2. **سجّل الدخول**
3. **أنشئ مستودع جديد** (Repository)
4. **اضغط على "Add file"**
5. **اختر "Upload files"**
6. **اختر الملفات من جهازك**:
   - index.html
   - styles.css
   - README.md
   - .gitignore
7. **اكتب رسالة الـ Commit**: "Initial commit"
8. **اضغط "Commit changes"**

---

## الخطوات المفصلة للطريقة 4 (الأسهل):

### 1. إنشاء المستودع:
```
github.com → + → New repository
Repository name: naseem-clinic-website
Visibility: Public
Create repository
```

### 2. رفع الملفات:
```
Add file → Upload files
اختر الملفات من جهازك
Commit changes
```

### 3. الحصول على الرابط:
بعد الرفع، رابط المستودع سيكون:
```
https://github.com/hadihaker56/naseem-clinic-website
```

## استكشاف الأخطاء:

**مشكلة**: لا أستطيع اختيار الملفات
**الحل**: استخدم تطبيق GitHub الرسمي بدلاً من المتصفح

**مشكلة**: كلمة المرور لا تعمل
**الحل**: استخدم Personal Access Token بدلاً من كلمة المرور

**مشكلة**: الملفات لم تُرفع بشكل صحيح
**الحل**: تأكد من أن الملفات بصيغة صحيحة (.html, .css, .md)

---

**الخلاصة**: أسهل طريقة هي استخدام تطبيق GitHub الرسمي أو موقع GitHub من متصفح الجوال!
