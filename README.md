# عيادات النسيم السنية التجميلية - الموقع الإلكتروني

موقع ويب حديث وفاخر لعيادات النسيم السنية التجميلية في دمشق، مصمم بأسلوب احترافي يعكس الهوية البصرية الراقية للعيادة.

## المميزات

- **تصميم مستجيب (Responsive Design)**: يعمل بشكل مثالي على جميع الأجهزة (الهاتف، الجهاز اللوحي، الكمبيوتر)
- **واجهة مستخدم سهلة**: تصميم بديهي وسهل الاستخدام
- **ألوان فاخرة**: استخدام ألوان احترافية تعكس الهوية الفاخرة
- **محتوى شامل**: يتضمن معلومات عن الخدمات، الأعمال، آراء العملاء، وطرق التواصل
- **تحسين محركات البحث (SEO)**: مُحسّن للظهور في نتائج البحث
- **سهل التخصيص**: يمكن تعديل المحتوى والألوان بسهولة

## الأقسام الرئيسية

1. **الرئيسية (Hero Section)**: عرض جذاب للعيادة مع دعوة للعمل
2. **عن العيادة**: معلومات عن العيادة والمميزات الرئيسية
3. **الخدمات**: عرض شامل لجميع الخدمات المقدمة
4. **الأعمال (Gallery)**: عرض الأعمال السابقة والنتائج
5. **آراء العملاء (Testimonials)**: شهادات من العملاء الراضين
6. **التواصل (Contact)**: نموذج التواصل ومعلومات الاتصال
7. **دعوة للعمل (CTA)**: زر واتساب والهاتف للحجز السريع

## التكنولوجيا المستخدمة

- **HTML5**: لبناء هيكل الصفحة
- **CSS3**: للتصميم والتنسيق
- **JavaScript**: للتفاعلات البسيطة
- **Font Awesome**: للأيقونات

## كيفية الاستخدام

### 1. استنساخ المستودع
```bash
git clone https://github.com/yourusername/naseem-clinic-website.git
cd naseem-clinic-website
```

### 2. فتح الموقع
ببساطة افتح ملف `index.html` في متصفحك المفضل.

### 3. التخصيص
يمكنك تعديل المحتوى والألوان من خلال:
- **HTML**: تعديل النصوص والمحتوى في `index.html`
- **CSS**: تعديل الألوان والتصميم في `styles.css`
